# 🛒 ( PRIONA - LiveShop ) Fullstack eCommerce Platform

A modern, scalable fullstack eCommerce web application built using **React, Redux, Node.js, Express, MongoDB, Socket.IO, and Firebase**. This project demonstrates a production-ready architecture with authentication, real-time communication, secure payments, and performance optimization.

---

## 🚀 Overview

This platform enables users to browse products, manage carts, place orders, make secure payments, and communicate in real-time. It also provides an admin system for managing products, users, and orders efficiently.

---

## ⚙️ Tech Stack

### 🎨 Frontend

* React.js for building dynamic user interfaces
* React Router for navigation (BrowserRouter, Nested Routes, Dynamic Routes, Protected Routes)
* Tailwind CSS for styling
* shadcn/ui for reusable UI components
* Framer Motion for smooth animations
* Redux Toolkit for global state management

---

### ⚙️ Backend

* Node.js runtime environment
* Express.js for building REST APIs
* MongoDB as the database
* Mongoose for schema modeling and database interaction

---

## 🔐 Authentication & Security
